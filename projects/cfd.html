CFD Heat Transfer Solver

Finite difference methods for heat conduction with ablation modeling

Overview

A comprehensive Python package for solving heat conduction problems using finite difference methods. Features include 1D and 2D steady-state solvers, transient heat conduction, and advanced 3D ablative heat shield modeling with material sublimation physics.

Key Features

  • 1D and 2D Steady-State Solvers: Solve Poisson equation for heat conduction with Dirichlet and Neumann boundary conditions
  • Transient Heat Conduction: Time-dependent 2D heat transfer with proper boundary condition handling
  • 3D Heat Shield Simulations: Cylindrical coordinate solver for reentry heat shield analysis
  • Ablative Material Modeling: Tracks material sublimation, latent heat accumulation, and surface recession
  • Grid Convergence: Source terms properly scaled for consistent results across different grid resolutions
  • Boundary Conditions: Support for both Dirichlet (fixed temperature) and Neumann (fixed flux) boundary conditions
  • Visualization: 2D animated plots, cylindrical plots, and r-z cross-sections for 3D analysis

Technical Details

The solver implements finite difference methods with proper handling of boundary conditions, grid convergence, and material phase changes. The ablation model tracks latent heat accumulation and material removal as elements reach sublimation temperature.

Key technical achievements:

  • Implemented dynamic surface tracking for ablative materials
  • Developed energy balance model for sublimation process
  • Created adaptive boundary condition handling for receding surfaces
  • Ensured grid convergence through proper source term scaling

Usage Example

from cfd.solvers.ablative_heat_shield import solve_heat_shield_3d

r, theta, z, transient = solve_heat_shield_3d(
    Nr=50, Ntheta=50, Nz=50, R=80, thickness=0.05,
    time=150, dt=1, k=0.04, rho=270, cp=1100,
    initial_temp=0, sub_temp=3500, sub_heat=2500000
)
